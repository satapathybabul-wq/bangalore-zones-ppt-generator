  BANGALORE ZONES POWERPOINT GENERATOR - QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════

WHAT THIS DOES:
Automatically generates a professional PowerPoint presentation with all Bangalore 
city zones organized by East, West, North, South, and Central with complete area 
listings for your network POP location connectivity diagram.

═══════════════════════════════════════════════════════════════════════════
SETUP (Windows/Mac/Linux):
═══════════════════════════════════════════════════════════════════════════

1. OPEN TERMINAL/COMMAND PROMPT
   Navigate to the repository folder

2. INSTALL REQUIREMENTS
   Command: pip install -r requirements.txt
   
   This installs the python-pptx library needed to generate PowerPoint files.

3. RUN THE GENERATOR
   Command: python generate_ppt.py
   
   Wait for the message: "✅ PowerPoint generated successfully!"

4. DOWNLOAD YOUR FILE
   Look for: Bangalore_Zones_Network_Diagram.pptx
   
   This is your ready-to-use PowerPoint presentation!
